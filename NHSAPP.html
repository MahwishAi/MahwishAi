NHS Vaccination Locator App with Integrated Google Maps and SQL Database

I created an NHS Vaccination Locator app in Java using the Eclipse IDE, aimed at helping users quickly find the nearest NHS vaccination center. The app features an intuitive, user-friendly interface built with Swing and incorporates the Google Maps API for precise location mapping. Users can search for the closest centers by entering a city, postcode, or specific location query, making it highly versatile and easy to use. The app is backed by an SQL database for efficient data management and reliable performance, ensuring users receive accurate and up-to-date information on nearby vaccination centers.
