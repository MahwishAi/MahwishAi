Advanced Position Stabilization with PID Cascade Control

In my project on unmanned aerial systems (UAS) control, I developed a feedback control algorithm for 2D position stabilization using a PID cascade control approach within a simulated environment. The system maintained the UAS at specified positions by processing positional targets, with a multi-layered PID design allowing independent tuning for vertical and horizontal adjustments. This setup enabled robust stabilization even under simulated environmental disturbances, such as gusty winds. Through iterative tuning and testing, the controller achieved reliable 2D stabilization, underscoring my proficiency in feedback control design and tuning essential for high-precision aerial robotics applications.

Reinforcement Learning Integration for Adaptive Control

To push the boundaries of control beyond conventional methods, I integrated reinforcement learning using OpenAI Gym and the Proximal Policy Optimization (PPO) algorithm after completing the PID control. This addition allowed the UAS to adaptively refine its performance in the simulated 2D environment by learning from previous actions and optimizing its responses. The reinforcement learning model helped the UAS to improve stability and positional accuracy over time, even when exposed to random target points and external gusts. This project not only expanded my skills in reinforcement learning applications within robotics but also showcased my ability to blend classical control theory with AI techniques for innovative, adaptive control solutions.
